Можно как-нибудь нумерацию картинок сделать как 1.1, 1.2...2.1, 2.2, 2.3..?

```typst
#set heading(numbering: "1.1")

#set figure(
  numbering: it => {
    let sec = counter(heading).at(here()).first()
    let fig = counter(figure).at(here()).first()
    return numbering("1.1", sec, fig)
  },
  supplement: [Рис.]
)

#show heading: it => {
  counter(figure).update(0)
  it
}
```

К слову оно все подряд приложения нумерует, а не картинки отдельно, таблицы отдельно
И там похоже нет адекватного красивого способа это разделить...
