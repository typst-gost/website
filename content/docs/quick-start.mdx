Шаблон `modern-g7-32` является пакетом для системы вёрстки Typst. Вы можете работать с ним двумя основными способами: в облачном редакторе (рекомендуется для новичков) или локально на своем компьютере (для продвинутых пользователей).

## Веб-приложение (Typst App)

Самый быстрый способ начать работу — использовать официальное веб-приложение. Оно не требует установки программного обеспечения и работает прямо в браузере.

**Преимущества:**
*   Не нужно устанавливать шрифты и компилятор.
*   Совместная работа над документом в реальном времени.
*   Доступ к документу с любого устройства.

### Как начать

1.  Зарегистрируйтесь или войдите на сайт [typst.app](https://typst.app).
2.  Нажмите кнопку **Create project** (Создать проект).
3.  Выберите вкладку **Template** (Шаблон) и в поиске введите `modern-g7-32`.
4.  Выберите шаблон и придумайте название для своего проекта.

{/* ![Интерфейс создания проекта в Typst App](typst-app-create.png) */}

После создания вы увидите редактор кода слева и предпросмотр документа справа. Вы можете сразу приступать к редактированию файла `main.typ`.

## Локальное использование

Для работы на собственном компьютере рекомендуется использовать редактор VS Code вместе с расширением Tinymist. Это обеспечивает максимальную производительность и возможность работы офлайн.

**Преимущества:**
*   Работа без интернета.
*   Использование системы контроля версий (Git).
*   Возможность автоматизации сборки (CI/CD).

### Настройка окружения

1.  Установите [VS Code](https://code.visualstudio.com/).
2.  Установите расширение **[Tinymist](https://marketplace.visualstudio.com/items?itemName=myriad-dreamin.tinymist)**. Это официальный инструмент для поддержки языка Typst (подсветка синтаксиса, автодополнение, живой предпросмотр).
3.  (Опционально) Установите [шрифты](https://github.com/typst-g7-32/modern-g7-32#шрифты), если хотите точное соответствие отображения. Typst умеет загружать шрифты из системы, но веб-версия использует встроенные.

### Создание проекта

Создайте файл `main.typ` в пустой папке и вставьте код подключения шаблона:

```typst
#import "@preview/modern-g7-32:0.2.0": gost

#show: gost.with(
  // Ваши настройки...
)

= Введение
```

Затем откройте палитру команд VS Code (`Ctrl+Shift+P` или `F1`) и выберите **Tinymist: Open Preview**. Справа откроется панель с вашим отчётом, которая будет обновляться при каждом сохранении файла.

{/*![VS Code с запущенным расширением Tinymist](vscode-tinymist.png)*/}

<Callout type="warning">
  При локальной работе Typst скачивает пакет шаблона при первой компиляции. Убедитесь, что у вас есть доступ к интернету в момент первого запуска. Пакеты кэшируются локально.
</Callout>

### Использование CLI

Если вы предпочитаете командную строку, вы можете установить [Typst CLI](https://github.com/typst/typst) и инициализировать проект одной командой:

```bash
typst init @preview/modern-g7-32
```

Эта команда создаст папку с готовой структурой файлов и примером использования.
